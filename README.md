# Search & Filter Web Application

## Live Demo
Deployed on Vercel: https://trend-pulse-website.vercel.app/

## Overview
A frontend web application that allows users to search and filter content dynamically using keyword input. The app fetches data from a backend API and renders results in real time.

The project focuses on practical JavaScript skills including API integration, DOM manipulation, asynchronous requests, and input validation.

## Features
- Real-time keyword search
- Fetches data from backend API
- Dynamic DOM rendering of results
- Input validation (length + empty input checks)
- Case-insensitive search handling
- Enter key support for faster interaction
- Live status updates during search

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Backend API (Node.js)
- Vercel (deployment)

## How It Works
1. User enters a search keyword
2. Input is validated and normalized
3. Frontend sends request to API endpoint:
4. Backend returns matching results
5. Frontend renders results dynamically in the UI

## Deployment
This project is deployed using **Vercel** for fast frontend hosting and automatic updates on push.
